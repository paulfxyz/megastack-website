The Stack

Nine tools.
One coherent fortress.

Every component is open source, auditable and connected to the others. No black boxes. No subscription traps. No calling home.

WireGuard
Layer 1 — VPN Backbone

The backbone of your entire stack. All other services tunnel through it. Modern cryptography, kernel-level performance, and a codebase small enough to actually audit. Every device, every server, one mesh.

WireGuard/wireguard-linux
Hoodik
Layer 2 — Encrypted File Hosting

End-to-end encrypted file storage on your own server. Store SSH keys, certificates, credentials and sensitive documents. Accessible only over WireGuard — invisible to the outside world.

hudikhq/hoodik
Hardened Firewall
Layer 3 — Network Policy

Custom nftables / iptables policies that ensure Hoodik, Matrix and all internal services are unreachable unless you're inside the WireGuard mesh. Deny-all defaults. Principle of least privilege, applied.

megastack-sh/firewall-templates
Access Management Templates
Layer 4 — Secure Operations

Production-ready Notion templates (and Plane.so equivalents) for tracking access grants, SSH key rotation, incident logs and VPN peer lists. All references point inside your WireGuard + Hoodik environment.

megastack-sh/access-templates
Matrix + Element
Layer 5 — Secure Communications

Self-hosted Matrix.org homeserver, federated only within your mesh. Use Element.io as your client for end-to-end encrypted team messaging, voice and file sharing. Zero metadata leakage to third-party servers.

element-hq/synapse
2FA Token Server
Layer 6 — Authentication

A self-hosted TOTP server (Authelia or Lldap + oath-toolkit) so your 2FA tokens never flow through Google Authenticator or Authy. Hardware key support (YubiKey / FIDO2) optional. Runs entirely inside your mesh.

authelia/authelia
Resilio Sync
Layer 7 — P2P File Sync

BitTorrent-based file sync across all your devices and servers. No central server, no third-party storage. Files replicate peer-to-peer, encrypted in transit. Works seamlessly through the WireGuard tunnel.

megastack-sh/resilio-config
topgrade
Layer 8 — Universal Updates

A single command that updates everything — Homebrew, apt, pip, cargo, snap, firmware, App Store apps — on every Mac, Windows machine and Linux server in your fleet. Custom fork with your stack's packages pre-configured.

paulfxyz/topgrade
Wasabi Wallet
Layer 9 — Bitcoin Privacy Reserve

Custom Wasabi Wallet setup for your Bitcoin reserve. Built-in CoinJoin for transaction privacy, Tor routing by default, and full-node support via your own Bitcoin Core. Non-custodial, open source — your keys never leave your machine, your transactions don't fingerprint you.

WalletWasabi/WalletWasabi

Runtime Security

Taps directly into Linux kernel events to detect anomalies in real time — unexpected network connections, container privilege escalations, suspicious file access. Catches what static scanning misses: threats already inside the perimeter.
