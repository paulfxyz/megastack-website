The Stack

Nine tools.
One coherent fortress.

Every component is open source, auditable and connected to the others. No black boxes. No subscription traps. No calling home.

WireGuard
Layer 1 — VPN Backbone

The backbone everything else runs through. Kernel-level performance, mathematically sound cryptography, and a codebase so lean you can actually read it. Every device, every server, one encrypted mesh — built to last.

WireGuard/wireguard-linux
Hoodik
Layer 2 — Encrypted File Hosting

End-to-end encrypted file storage on your own server. SSH keys, certificates, credentials, sensitive documents — locked and accessible only inside the WireGuard mesh. To the outside world, it does not exist.

hudikhq/hoodik
Hardened Firewall
Layer 3 — Network Policy

Custom nftables policies enforcing a single rule: if you are not inside the mesh, you see nothing. Deny-all defaults. Every exception is explicit, documented and auditable. Least privilege at the network layer.

megastack-sh/firewall-templates
Access Management Templates
Layer 4 — Secure Operations

Operational playbooks for running a secure infrastructure: key rotation schedules, peer onboarding checklists, incident timelines, access revocation flows. Everything links back into your own environment — nothing lives on a third-party platform.

megastack-sh/access-templates
Matrix + Element
Layer 5 — Secure Communications

Self-hosted Matrix homeserver, federated only within your WireGuard mesh. E2E encrypted messaging, voice calls and file transfer via Element — with zero metadata touching any external server. Your comms, fully owned.

element-hq/synapse
2FA Token Server
Layer 6 — Authentication

Your 2FA tokens should never pass through Google, Authy or any third party. Authelia gives you a self-hosted authentication gateway with TOTP, hardware key support (YubiKey, FIDO2) and SSO — running entirely inside your mesh.

authelia/authelia
Resilio Sync
Layer 7 — P2P File Sync

BitTorrent-based sync with no central server and no cloud dependency. Files move peer-to-peer between your devices, encrypted in transit, routed through the WireGuard tunnel. Fast, private, decentralised by design.

megastack-sh/resilio-config
topgrade
Layer 8 — Universal Updates

One command. Every package manager, every platform — Homebrew, apt, pip, cargo, winget, firmware — updated across your entire fleet simultaneously. The only maintenance task that actually stays done.

paulfxyz/topgrade
Wasabi Wallet
Layer 9 — Bitcoin Privacy Reserve

Non-custodial Bitcoin infrastructure for your reserve. CoinJoin breaks transaction traceability, Tor routes your network traffic, your own Bitcoin Core node removes any third-party trust. Your keys, your coins — untrackable by design.

WalletWasabi/WalletWasabi

Runtime Security

Taps directly into Linux kernel events to detect anomalies in real time — unexpected network connections, container privilege escalations, suspicious file access. Catches what static scanning misses: threats already inside the perimeter.

Static Code Analysis

Rapid multi-language static analysis with custom rule support to enforce organisation-specific security patterns. Runs in CI — catches insecure coding patterns, injection vectors and dangerous function calls before any code ships.
