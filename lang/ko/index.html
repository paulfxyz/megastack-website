작동 방식

세 단계.
완전한 주권.

SaaS 없음. 벤더 락인 없음. 신뢰 전제 없음 — 당신이 읽을 수 있는 코드, 당신이 통제하는 인프라뿐입니다.

01
진단

60분의 보안 통화에서 당신의 위협 모델, 기존 인프라, 팀의 운영 습관을 함께 정리합니다. 똑같은 스택은 두 개가 없습니다.

02
배포

당신이 이미 소유하거나 통제하는 서버 위에 모든 컴포넌트를 올립니다. 강화된 기본값, 방화벽 정책, 레이어 간 통합 — 전부 자동화, 전부 오픈소스.

03
인수인계

전체 문서, 관리자 권한, 모든 기기에서 도는 topgrade까지 받게 됩니다. 우리는 빠집니다 — 스택은 우리 없이도 굴러갑니다. 원한다면 분기별 감사를 위해 우리를 계속 쓸 수도 있습니다.

스택

툴 9종.
하나로 짜인 요새.

모든 컴포넌트는 오픈소스이고, 감사 가능하며, 서로 맞물려 있습니다. 블랙박스 없음. 구독 함정 없음. 몰래 외부 호출하는 일도 없음.

WireGuard
Layer 1 — VPN 백본

다른 모든 것이 그 위에서 돌아가는 백본. 커널 레벨의 성능, 수학적으로 빈틈없는 암호 체계, 그리고 사람이 실제로 다 읽을 수 있을 만큼 군더더기 없는 코드베이스. 모든 기기, 모든 서버가 하나의 암호화 메시 위에 — 오래 가도록 만들어졌습니다.

WireGuard/wireguard-linux
Hoodik
Layer 2 — 암호화 파일 호스팅

당신 자신의 서버에서 돌아가는 종단 간 암호화 스토리지. SSH 키, 인증서, 자격 증명, 민감 문서들 — 잠겨 있고, 오직 WireGuard 메시 안에서만 접근 가능합니다. 외부 세계에서는 아예 존재하지 않는 것이나 마찬가지입니다.

hudikhq/hoodik
Hardened Firewall
Layer 3 — 네트워크 정책

딱 하나의 규칙을 강제하는 커스텀 nftables 정책: 메시 안에 있지 않으면, 아무것도 보이지 않는다. 기본은 전부 거부. 모든 예외는 명시적이고, 문서화되어 있고, 감사 가능합니다. 네트워크 계층에서의 최소 권한 원칙.

megastack-sh/firewall-templates
Access Management Templates
Layer 4 — 보안 운영

보안 인프라를 굴리기 위한 운영 플레이북: 키 로테이션 일정, 피어 온보딩 체크리스트, 사고 타임라인, 권한 회수 절차. 모든 것이 당신 자신의 환경으로 회귀합니다 — 어떤 것도 외부 플랫폼에 남지 않습니다.

megastack-sh/access-templates
Matrix + Element
Layer 5 — 보안 통신

셀프호스트 Matrix homeserver, 페더레이션은 오직 당신의 WireGuard 메시 안에서만. Element를 통한 E2E 암호화 메시지, 음성 통화, 파일 전송 — 메타데이터 한 조각도 외부 서버로 새지 않습니다. 통신은 온전히 당신 것.

element-hq/synapse
2FA Token Server
Layer 6 — 인증

당신의 2FA 토큰이 Google이나 Authy, 그 어떤 제3자를 거칠 이유는 없습니다. Authelia는 TOTP, 하드웨어 키(YubiKey, FIDO2), SSO를 지원하는 셀프호스트 인증 게이트웨이를 — 메시 안에서 완결되게 — 제공합니다.

authelia/authelia
Resilio Sync
Layer 7 — P2P 파일 동기화

BitTorrent 기반 동기화 — 중앙 서버 없음, 클라우드 의존 없음. 파일은 당신의 기기들 사이에서 P2P로 오가고, 전송 구간이 암호화되며, 경로는 WireGuard 터널을 탑니다. 빠르고, 사적이고, 설계상 분산형입니다.

megastack-sh/resilio-config
topgrade
Layer 8 — 통합 업데이트

명령 한 줄. 모든 패키지 매니저, 모든 플랫폼 — Homebrew, apt, pip, cargo, winget, 펌웨어 — 가 당신의 전체 디바이스에서 동시에 업데이트됩니다. 실제로 '끝낼 수 있는' 유일한 유지보수 작업.

paulfxyz/topgrade
Wasabi Wallet
Layer 9 — 비트코인 프라이버시 리저브

당신의 리저브를 위한 논커스터디얼 비트코인 인프라. CoinJoin이 트랜잭션 추적성을 끊고, Tor가 네트워크 트래픽을 익명화하며, 직접 운용하는 Bitcoin Core 노드가 모든 제3자 신뢰를 제거합니다. 당신의 키, 당신의 코인 — 설계 자체가 추적 불가입니다.

WalletWasabi/WalletWasabi
확장 툴박스

코어 스택 너머.
