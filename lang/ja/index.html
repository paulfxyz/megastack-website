仕組み

3 ステップ。
完全な主権。

SaaS なし。ベンダーロックインなし。信頼を前提にしない——あなたが読めるコードと、あなたが操れるインフラだけ。

01
アセスメント

60 分のセキュアな通話で、あなたの脅威モデル、既存インフラ、チームの運用習慣を整理します。同じスタックは二つとありません。

02
デプロイ

あなたがすでに所有・管理しているサーバーに、各コンポーネントを構築します。堅牢なデフォルト、ファイアウォールポリシー、レイヤー間の連携——すべて自動化、すべてオープンソース。

03
引き渡し

完全なドキュメント、管理者アクセス、各デバイスで動く topgrade をお渡しします。私たちはここで離脱——スタックは私たちなしで動き続けます。希望があれば四半期ごとの監査だけ継続できます。

スタック

ツール 9 種。
一体型の要塞。

すべてのコンポーネントはオープンソース、監査可能、そして互いに連動しています。ブラックボックスなし。サブスク地獄なし。勝手に外部と通信したりもしません。

WireGuard
Layer 1 — VPN バックボーン

他のすべてが上に乗る基幹レイヤー。カーネルレベルのパフォーマンス、数学的に隙のない暗号、人が実際に読み切れるほど引き締まったコードベース。すべての端末、すべてのサーバーが、ひとつの暗号化メッシュに——長く使うために設計されています。

WireGuard/wireguard-linux
Hoodik
Layer 2 — 暗号化ファイルホスティング

あなた自身のサーバー上で動くエンドツーエンド暗号化ストレージ。SSH キー、証明書、認証情報、機密ドキュメント——施錠され、WireGuard メッシュの内側からしかアクセスできません。外の世界からは存在しないも同然です。

hudikhq/hoodik
Hardened Firewall
Layer 3 — ネットワークポリシー

カスタム nftables ポリシーがたったひとつのルールを徹底します：メッシュの内側にいなければ、何も見えない。デフォルトはすべて拒否。例外はすべて明示・文書化・監査可能。ネットワーク層での最小権限の原則。

megastack-sh/firewall-templates
Access Management Templates
Layer 4 — セキュアな運用

セキュアなインフラを回すための運用プレイブック：鍵ローテーション計画、ピアのオンボーディングチェックリスト、インシデントタイムライン、アクセス剥奪フロー。すべてはあなた自身の環境に紐づき、サードパーティのプラットフォームには何も残しません。

megastack-sh/access-templates
Matrix + Element
Layer 5 — セキュア通信

セルフホストの Matrix homeserver。フェデレーションは WireGuard メッシュ内のみ。Element 経由で E2E 暗号化されたメッセージ、音声通話、ファイル転送——メタデータが外部サーバーに漏れることは一切ありません。通信は完全にあなたのもの。

element-hq/synapse
2FA Token Server
Layer 6 — 認証

あなたの 2FA トークンが Google や Authy、その他のサードパーティを経由していい理由はありません。Authelia は、TOTP、ハードウェアキー（YubiKey、FIDO2）、SSO に対応したセルフホストの認証ゲートウェイを、メッシュの内側で完結して提供します。

authelia/authelia
Resilio Sync
Layer 7 — P2P ファイル同期

BitTorrent ベースの同期——中央サーバーなし、クラウド依存なし。ファイルはあなたの端末同士でピアツーピアにやり取りされ、転送中も暗号化、経路は WireGuard トンネル。速くて、プライベートで、設計レベルで分散型です。

megastack-sh/resilio-config
topgrade
Layer 8 — 横断アップデート

コマンドひとつ。Homebrew、apt、pip、cargo、winget、ファームウェア——あらゆるパッケージマネージャーとプラットフォームが、あなたの全端末で同時に更新されます。実際に「やり切れる」ただひとつのメンテナンスタスク。

paulfxyz/topgrade
Wasabi Wallet
Layer 9 — ビットコインのプライバシーリザーブ

あなたのリザーブ向けのノンカストディアルなビットコイン基盤。CoinJoin がトランザクションの追跡可能性を断ち切り、Tor がネットワーク通信を匿名化、自前の Bitcoin Core ノードがサードパーティへの信頼を完全に排除します。あなたの鍵、あなたのコイン——設計上、追跡されません。

WalletWasabi/WalletWasabi

OpenVAS / Greenbone
脆弱性スキャン

幅広いプロトコル領域とコンプライアンスフレームワークをカバーし、毎日更新されるチェックで網羅的に脆弱性をスキャン。認証付きスキャンが、悪用される前に設定ミスを拾い上げます。チケッティングシステムと連携し、修正までの流れもスムーズに。

greenbone/openvas-scanner

Trivy
コンテナ・クラウドセキュリティ

コンテナ、Kubernetes、IaC テンプレート、クラウド設定までを一つのツールでカバーする万能スキャナ。脆弱性や設定ミスを本番に届く前に検出——CI/CD パイプラインに直接組み込めます。

aquasecurity/trivy

Falco
ランタイムセキュリティ

Linux カーネルのイベントに直接フックし、想定外のネットワーク接続、コンテナの権限昇格、不審なファイルアクセスといった異常をリアルタイムで検知。静的スキャンが取り逃すもの——すでに境界の内側にいる脅威を捕まえます。

falcosecurity/falco

Semgrep
静的コード解析

高速なマルチ言語の静的解析。カスタムルールに対応しているので、組織固有のセキュリティパターンを徹底できます。CI で動かして、危険なコードパターン、インジェクションの入口、危ない関数呼び出しを、デプロイ前に潰します。

semgrep/semgrep

Gitleaks
シークレット検出

git の全履歴をスキャンし、流出した API キー、認証情報、トークン、秘密値を本番に届く前に発見——あるいはすでにそこにあるものを見つけ出します。pre-commit と CI に組み込み可能。最も安易な侵入経路を、ふさぐ。

gitleaks/gitleaks
