การสื่อสารของคุณ Stack ของคุณ

เราติดตั้ง stack ความปลอดภัยแบบ open source ที่ผ่านศึกมาอย่างหนัก ครบทุกชิ้น บนโครงสร้างพื้นฐานที่คุณเป็นเจ้าของ ทั้ง VPN, ที่เก็บไฟล์เข้ารหัส, การส่งข้อความปลอดภัย, 2FA, sync และ auto-update — ตั้งค่าให้ทำงานร่วมกันเป็นหนึ่งเดียว

ดู stack

⚡ทุกอย่างทำงานบนเซิร์ฟเวอร์ของคุณ — เราไม่แตะข้อมูลคุณอีกเลยหลังติดตั้งเสร็จ⚡

วิธีทำงาน

สามขั้นตอน
อธิปไตยเต็มรูปแบบ

ไม่มี SaaS ไม่มี vendor lock-in ไม่ต้องไว้ใจใคร — มีแค่โค้ดที่คุณอ่านได้ และโครงสร้างที่คุณคุมเอง

01
ประเมิน

เราใช้เวลา 60 นาทีในการคุยแบบปลอดภัย เพื่อแมป threat model ของคุณ โครงสร้างที่มีอยู่ และพฤติกรรมการทำงานของทีม ไม่มี stack ไหนเหมือนกันเป๊ะ

02
ติดตั้ง

เราติดตั้งทุกองค์ประกอบลงบนเซิร์ฟเวอร์ที่คุณเป็นเจ้าของหรือควบคุมอยู่แล้ว ค่าตั้งต้นที่แข็งแกร่ง, นโยบาย firewall, การเชื่อมต่อระหว่างชั้น — ทำอัตโนมัติทั้งหมด open source ทั้งหมด

03
ส่งมอบ

คุณจะได้เอกสารครบ สิทธิ์ admin และ topgrade รันอยู่ในทุกอุปกรณ์ เราถอนตัว — stack ของคุณยังเดินต่อโดยไม่มีเรา หากต้องการ ก็จ้างเราเป็นรายไตรมาสไว้ตรวจสอบได้

The Stack

เก้าเครื่องมือ
ป้อมปราการเดียวที่ลงตัว

ทุกองค์ประกอบเป็น open source ตรวจสอบได้ และเชื่อมโยงเข้าหากัน ไม่มีกล่องดำ ไม่มีกับดักสมาชิก ไม่มีการส่งข้อมูลกลับบ้าน

WireGuard
ชั้นที่ 1 — VPN Backbone

กระดูกสันหลังที่ทุกอย่างวิ่งผ่าน ประสิทธิภาพระดับ kernel, การเข้ารหัสที่แม่นยำเชิงคณิตศาสตร์, codebase กระชับจนคุณอ่านได้จริง ทุกอุปกรณ์ ทุกเซิร์ฟเวอร์ รวมเป็น mesh เข้ารหัสเดียว — สร้างมาเพื่ออยู่ยาว

WireGuard/wireguard-linux
Hoodik
ชั้นที่ 2 — ที่เก็บไฟล์เข้ารหัส

ที่เก็บไฟล์เข้ารหัส end-to-end บนเซิร์ฟเวอร์ของคุณเอง กุญแจ SSH, ใบรับรอง, credentials, เอกสารสำคัญ — ล็อกแน่น เข้าได้เฉพาะภายใน mesh ของ WireGuard เท่านั้น สำหรับโลกภายนอก มันไม่มีอยู่จริง

hudikhq/hoodik
Hardened Firewall
ชั้นที่ 3 — Network Policy

นโยบาย nftables ที่ปรับแต่งเอง บังคับใช้กฎเดียว: ถ้าคุณไม่ได้อยู่ใน mesh คุณจะไม่เห็นอะไรเลย ค่าเริ่มต้นคือปฏิเสธทั้งหมด ทุกข้อยกเว้นต้องชัดเจน มีเอกสาร ตรวจสอบได้ Least privilege ที่ชั้นเครือข่าย

megastack-sh/firewall-templates
Access Management Templates
ชั้นที่ 4 — การปฏิบัติการอย่างปลอดภัย

Playbook สำหรับการรันโครงสร้างที่ปลอดภัย: ตารางหมุนกุญแจ, checklist รับ peer ใหม่, ไทม์ไลน์รับมือเหตุการณ์, ขั้นตอนเพิกถอนสิทธิ์ ทุกอย่างผูกกลับเข้าสภาพแวดล้อมของคุณเอง — ไม่มีอะไรอยู่บนแพลตฟอร์มของบุคคลที่สาม

megastack-sh/access-templates
Matrix + Element
ชั้นที่ 5 — การสื่อสารปลอดภัย

Matrix homeserver แบบ self-host federate เฉพาะใน mesh ของ WireGuard ของคุณ ส่งข้อความ โทรเสียง และส่งไฟล์เข้ารหัส E2E ผ่าน Element — โดยไม่มี metadata แตะเซิร์ฟเวอร์ภายนอกเลย การสื่อสารของคุณ เป็นของคุณเต็มที่

element-hq/synapse
2FA Token Server
ชั้นที่ 6 — การยืนยันตัวตน

โทเค็น 2FA ของคุณไม่ควรวิ่งผ่าน Google, Authy หรือบุคคลที่สาม Authelia ให้เกตเวย์ยืนยันตัวตนแบบ self-host พร้อม TOTP, รองรับกุญแจฮาร์ดแวร์ (YubiKey, FIDO2) และ SSO — รันภายใน mesh ของคุณทั้งหมด

authelia/authelia
Resilio Sync
ชั้นที่ 7 — Sync ไฟล์แบบ P2P

Sync แบบ BitTorrent ไม่มีเซิร์ฟเวอร์กลาง และไม่พึ่ง cloud ไฟล์เคลื่อนที่ peer-to-peer ระหว่างอุปกรณ์ของคุณ เข้ารหัสระหว่างทาง ส่งผ่านอุโมงค์ WireGuard เร็ว เป็นส่วนตัว กระจายศูนย์โดยการออกแบบ

megastack-sh/resilio-config
topgrade
ชั้นที่ 8 — อัปเดตทั้งระบบ

คำสั่งเดียว ทุก package manager ทุกแพลตฟอร์ม — Homebrew, apt, pip, cargo, winget, firmware — อัปเดตทั่วทั้งกองอุปกรณ์ของคุณพร้อมกัน งานบำรุงรักษาเดียวที่ทำเสร็จแล้วเสร็จจริง

paulfxyz/topgrade
Wasabi Wallet
ชั้นที่ 9 — เงินสำรอง Bitcoin แบบเป็นส่วนตัว

โครงสร้าง Bitcoin แบบ non-custodial สำหรับเงินสำรองของคุณ CoinJoin ทำลายการตามรอยธุรกรรม Tor ส่งทราฟฟิกเครือข่ายของคุณ และโหนด Bitcoin Core ของคุณเองตัดความไว้ใจในบุคคลที่สามทิ้งไป กุญแจของคุณ เหรียญของคุณ — ตามรอยไม่ได้โดยการออกแบบ

WalletWasabi/WalletWasabi

คำถามที่ควรถาม

ส่วนใหญ่ติดตั้งเสร็จใน 1–3 วัน ขึ้นอยู่กับจำนวนเซิร์ฟเวอร์และอุปกรณ์ การประเมินใช้เวลา 60 นาที หลังจากนั้น สคริปต์ provisioning อัตโนมัติของเราจะจัดการงานส่วนใหญ่ — คุณแค่ต้องให้สิทธิ์ root SSH แก่เราเพื่อเข้าเซิร์ฟเวอร์ของคุณ
ไม่ต้อง เราช่วยคุณ provision เซิร์ฟเวอร์ bare-metal หรือ VPS จากผู้ให้บริการอย่าง Hetzner, OVHcloud หรือ Vultr ได้ — ทั้งหมดอยู่ภายใต้บัญชีของคุณเอง หลักการสำคัญ: คุณถือ credentials เราไม่เคยถือ
คุณได้รับเอกสารครบ admin credentials และคอนฟิก topgrade ที่ใช้งานได้ stack ทำงานต่อโดยไม่มีเรา ตัวเลือก: คุณจ้างเราต่อสำหรับการตรวจสอบความปลอดภัยรายไตรมาส, อัปเดต stack หรือขยายการติดตั้งได้
ใช่ WireGuard, Hoodik, Matrix/Synapse, Authelia, Resilio และ topgrade ทั้งหมดเป็น open source อยู่บน GitHub เทมเพลต firewall และเทมเพลตการจัดการสิทธิ์ของเราก็เป็น open source ภายใต้ MIT คุณตรวจดูได้ทุกบรรทัดก่อนเราจะแตะโครงสร้างคุณ
การติดต่อครั้งแรกใช้ Signal หรือแพลตฟอร์มวิดีโอเข้ารหัสที่ลูกค้าเลือก หลัง onboarding เสร็จ การสื่อสารทั้งหมดย้ายไปที่ Matrix homeserver ของคุณเอง เราไม่คุยรายละเอียดลูกค้าผ่านอีเมล, Slack, Zoom หรือแพลตฟอร์มของบุคคลที่สามใดๆ
