Hoe het werkt

Drie stappen.
Volledige soevereiniteit.

Geen SaaS. Geen vendor lock-in. Geen vertrouwen nodig — gewoon code die je kunt lezen en infrastructuur die je beheert.

01
Analyseer

We besteden 60 minuten aan een veilig gesprek om je threat model, je bestaande infrastructuur en de werkgewoonten van je team in kaart te brengen. Geen twee stacks zijn identiek.

02
Uitrollen

Overdracht

De Stack

Negen tools.
Eén samenhangend fort.

Elk component is open source, controleerbaar en verbonden met de rest. Geen black boxes. Geen abonnementsvallen. Geen phone-home.

WireGuard
Laag 1 — VPN-ruggengraat

De ruggengraat waar al het andere doorheen loopt. Kernel-level performance, wiskundig solide cryptografie en een codebase zo strak dat je hem daadwerkelijk kunt lezen. Elk apparaat, elke server, één versleutelde mesh — gebouwd om te blijven.

WireGuard/wireguard-linux
Hoodik
Laag 2 — Versleutelde bestandsopslag

End-to-end versleutelde bestandsopslag op je eigen server. SSH-sleutels, certificaten, credentials, gevoelige documenten — opgeborgen en alleen toegankelijk binnen de WireGuard-mesh. Voor de buitenwereld bestaat het simpelweg niet.

hudikhq/hoodik
Hardened Firewall
Laag 3 — Netwerkbeleid

Aangepaste nftables-regels die één principe afdwingen: zit je niet in de mesh, dan zie je niets. Deny-all als standaard. Elke uitzondering is expliciet, gedocumenteerd en controleerbaar. Least privilege op netwerkniveau.

megastack-sh/firewall-templates
Access Management Templates
Laag 4 — Veilige operaties

Operationele playbooks voor het draaien van een veilige infrastructuur: schema's voor key rotation, onboarding-checklists voor peers, incidenttijdlijnen, flows voor toegangsintrekking. Alles linkt terug naar je eigen omgeving — niets leeft op een externe platform.

megastack-sh/access-templates
Matrix + Element
Laag 5 — Veilige communicatie

Self-hosted Matrix-homeserver, alleen gefedereerd binnen je WireGuard-mesh. E2E-versleutelde berichten, voicecalls en bestandsoverdracht via Element — zonder dat er metadata een externe server raakt. Je communicatie, volledig in eigen beheer.

element-hq/synapse
2FA Token Server
Laag 6 — Authenticatie

Je 2FA-tokens horen nooit door Google, Authy of een andere derde partij te lopen. Authelia geeft je een self-hosted authenticatiegateway met TOTP, ondersteuning voor hardware keys (YubiKey, FIDO2) en SSO — volledig draaiend binnen je mesh.

authelia/authelia
Resilio Sync
Laag 7 — P2P-bestandsync

BitTorrent-gebaseerde sync zonder centrale server en zonder cloudafhankelijkheid. Bestanden gaan peer-to-peer tussen je apparaten, versleuteld onderweg, gerouteerd door de WireGuard-tunnel. Snel, privé, gedecentraliseerd by design.

megastack-sh/resilio-config
topgrade
Laag 8 — Universele updates

Eén commando. Elke package manager, elk platform — Homebrew, apt, pip, cargo, winget, firmware — tegelijkertijd geüpdatet over je hele wagenpark. De enige onderhoudstaak die echt afgehandeld blijft.

paulfxyz/topgrade
Wasabi Wallet
Laag 9 — Bitcoin-privacyreserve

Non-custodial Bitcoin-infrastructuur voor je reserve. CoinJoin doorbreekt traceerbaarheid van transacties, Tor routeert je netwerkverkeer, je eigen Bitcoin Core-node verwijdert elk vertrouwen in derden. Jouw sleutels, jouw coins — by design niet te volgen.

WalletWasabi/WalletWasabi
Uitgebreide toolbox

Verder dan de kernstack.

Afhankelijk van je threat model voegen we extra open-source tools toe voor vulnerability scanning, runtime monitoring, code-analyse en intrusion detection. Elke opdracht is op maat — je krijgt alleen wat je nodig hebt.

🔍
OpenVAS / Greenbone
Vulnerability scanning

Uitgebreide vulnerability scanning met dagelijks bijgewerkte checks over een breed scala aan protocollen en compliance-frameworks. Geauthenticeerde scans vangen misconfiguraties op voor ze worden uitgebuit. Integreert met ticketingsystemen voor gestroomlijnde remediation.

greenbone/openvas-scanner
📦
Trivy
Container- & cloudbeveiliging

Veelzijdige scanner die containers, Kubernetes, IaC-templates en cloudconfiguraties dekt in één tool. Vindt vulnerabilities en misconfiguraties voor ze in productie belanden — direct geïntegreerd in CI/CD-pipelines.

aquasecurity/trivy
⚡
Falco
Runtime security

Tapt direct in op Linux-kernel events om afwijkingen in realtime te detecteren — onverwachte netwerkverbindingen, container privilege escalations, verdachte bestandsacties. Vangt op wat statische scans missen: dreigingen die al binnen de perimeter zitten.

falcosecurity/falco
🧬
Semgrep
Statische code-analyse

Snelle multi-language statische analyse met ondersteuning voor custom rules om organisatie-specifieke security patterns af te dwingen. Draait in CI — vangt onveilige codepatronen, injection vectors en gevaarlijke functieaanroepen op voordat er code live gaat.

semgrep/semgrep
🔑
Gitleaks
Secret detection

Scant je hele git-history op blootgestelde API-keys, credentials, tokens en secrets voor ze in productie komen — of vindt degene die er al zitten. Hookt in op pre-commit en CI. De makkelijkste breach vector, dichtgetimmerd.

gitleaks/gitleaks
🕸️
OWASP ZAP
Web app security testing

De meest volwassen open-source web application security scanner. Geautomatiseerde detectie van SQL-injection, XSS, broken authentication en tientallen andere OWASP Top 10-vulnerabilities. Actief en passief scannen, met CI-integratie.

zaproxy/zaproxy
